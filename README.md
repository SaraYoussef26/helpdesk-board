## Helpdesk Ticket Board

## Project Description

This is a Helpdesk Ticket Board built with Next.js and JSX. 

The app allows users to:
	•	Browse tickets loaded from an API
	•	Filter tickets by status and priority
	•	Search tickets by title or description
	•	Add tickets to My Queue (like a cart)
	•	See live updates to ticket status and priority
	•	View helpful messages for loading, errors, or empty states


## Setup
	1.	npm install
	2.	npm run dev
	3.	Open http://localhost:3000 in your browser
